Shopee 广告全流程说明

三个客服案例完成后，下一步做 Shopee 广告是合理的；它属于“流量获取 → 点击 → 转化 → 订单/成本/利润”的经营核心模块。

真实 Shopee 广告可以理解成：卖家花钱买曝光，把商品推到买家更容易看到的位置，然后按点击或投放消耗付费，最后看这些广告有没有带来订单。

1. Shopee 广告在卖家经营里的位置

一个店铺正常卖货大概有几条流量来源：

  1. 自然流量：买家搜索、分类浏览、推荐位自然看到你的商品。
  2. 营销活动流量：折扣、代金券、运费促销、限时抢购带来的流量。
  3. 广告流量：卖家主动付费，让商品获得更多曝光。
我的商品已经上架了，但没人看到，或者自然排名不够靠前，所以我要花预算买一些曝光和点击。

2. Shopee 广告的基本流程

第一步：进入 Shopee Ads / 广告中心

卖家进入广告后台，通常会看到广告总览、当前花费、点击量、曝光量、订单数、销售额、ROAS / 投产比、正在运行的广告活动。这里类似一个广告经营仪表盘。

第二步：选择广告类型

广告类型简单理解适合场景
搜索广告买家搜索关键词时，你的商品出现在靠前位置商品有明确搜索词，比如“手机壳”“卫生纸”
发现广告 / 推荐广告商品出现在推荐流、猜你喜欢、相似商品等位置想扩大曝光、拉新
商店广告推广整个店铺，而不是单个商品店铺商品多、品牌感强
关联广告 / 商品推荐位出现在其他商品详情页附近抢竞品流量或相似商品流量

不同站点名称可能略有不同，但核心逻辑差不多：买曝光和点击。

第三步：选择要推广的商品

广告通常是围绕商品来的。卖家要选推广哪个商品、商品是否已经上架、库存是否充足、商品评分/价格/图片是否有竞争力、是否适合投广告。

真实运营里，不是所有商品都适合投广告。更适合投广告的商品一般是：有库存、图片好、标题清楚、价格有竞争力、已经有销量或评价、利润空间能覆盖广告成本。

如果一个商品本身很差，投广告只是让更多人看到它，但不一定有人买。

第四步：设置预算

预算类型含义
每日预算每天最多花多少钱
总预算这个广告活动最多花多少钱

比如每日预算 RM20、总预算 RM200，那么系统每天最多烧 RM20，总共最多烧 RM200。如果预算烧完，广告就会停止展示或暂停。

第五步：设置出价

广告不是你设置了预算就一定能展示，通常还要出价。出价可以理解为：我愿意为一次点击最多付多少钱。

例如每次点击最高 RM0.20、RM0.50 或 RM1.00。出价越高，理论上越容易获得曝光和靠前位置，但成本也越高。

真实广告系统一般会综合出价、商品质量、点击率、转化率、关键词相关性和店铺表现，不是单纯谁出价高谁赢。

第六步：选择投放方式

模式含义适合谁
自动广告系统自动帮你选关键词/人群/展示位置新手、想快速开始
手动广告卖家自己选关键词、出价、匹配方式有经验、想精细控制成本

自动广告更简单：选商品、设预算，系统帮你跑。手动广告更像运营：我要投“卫生纸”“抽取式卫生纸”“便宜卫生纸”这些关键词，每个词给不同出价。

第七步：投放后看数据并优化

广告不是创建完就结束了，真正重要的是后续看数据。

指标含义怎么理解
曝光量广告被看到多少次有没有流量
点击量买家点了多少次商品图/标题/价格有没有吸引力
CTR 点击率点击量 ÷ 曝光量看到的人愿不愿意点
花费广告烧了多少钱成本
CPC平均每次点击多少钱点击成本
订单数广告带来多少订单是否转化
销售额广告带来的成交金额收入
ROAS销售额 ÷ 广告花费投产比
ACOS广告花费 ÷ 广告销售额成本占比

例如广告花了 RM10，带来销售额 RM50，ROAS = 5，表示每花 RM1 广告费，带来 RM5 销售额。如果商品毛利够高，这可能是赚的。

如果花了 RM10，只卖了 RM12，商品利润才 RM3，那广告可能是亏的。

3. 一个完整广告案例

假设你卖“抽取式卫生纸”。

创建前

  • 售价 RM12
  • 成本 RM7
  • 毛利 RM5
  • 库存 300
  • 主图还不错
  • 标题包含“抽取式卫生纸”
  • 有一点销量

这个商品可以试投。

创建广告

  • 推广商品：抽取式卫生纸
  • 每日预算：RM20
  • 投放时间：7 天
  • 关键词：卫生纸、抽取式卫生纸、家用卫生纸
  • 出价：RM0.20 - RM0.50 每次点击

跑了一天

  • 曝光：5,000
  • 点击：100
  • 花费：RM25
  • 订单：8
  • 销售额：RM96

CTR = 100 / 5000 = 2%，CPC = RM25 / 100 = RM0.25，ROAS = RM96 / RM25 = 3.84。如果商品每单毛利 RM5，8 单毛利 RM40，广告花 RM25，那还剩 RM15 毛利，广告是可接受的。

优化

  • 提高表现好的关键词出价
  • 降低只花钱不出单的关键词出价
  • 暂停无转化关键词
  • 给商品加折扣或代金券提高转化
  • 优化主图和标题提高点击率
  • 控制每日预算避免烧太快

4. 广告模块如果做进模拟系统，可以拆成哪些能力

后面做设计文档/代码时，可以考虑先做 V1，不要一开始做太复杂。

V1 推荐功能

模块功能
广告首页展示广告总花费、点击、订单、销售额、ROAS
创建广告选择商品、广告类型、预算、出价、投放时间
广告列表展示活动状态、预算、花费、点击、订单、销售额
暂停/继续/结束控制广告活动状态
广告模拟按游戏时间消耗预算，生成曝光、点击、转化
数据归因标记哪些订单来自广告
优化反馈提示“点击高但转化低”“预算耗尽”“ROAS 偏低”

V1 可以先不做

暂不做原因
复杂关键词匹配第一版会变得很重
人群画像投放需要大量用户行为模型
实时竞价系统对模拟教学不必要
多广告位精细竞价可以后续扩展
复杂归因窗口V1 可先用“点击后短时间内下单归因”

5. 在模拟系统里，Shopee 广告最好怎么理解

已有模块广告如何关联
我的商品只能选择已上架商品投广告
商品质量评分商品越好，广告点击率/转化率越高
库存没库存应暂停或降低广告效果
折扣/代金券有优惠时广告转化率更高
运费促销可能提高转化率
订单模拟广告带来额外曝光和订单概率
资金/收入广告花费应影响利润
经营统计展示广告花费、广告订单、ROAS
Shopee 广告不是单独的页面功能，而是一个会消耗预算、增加曝光、影响订单生成、最后改变利润的经营杠杆。

建议下一步

  1. 先做哪种广告：建议先做“商品搜索广告”。
  2. 广告是否真实影响订单模拟：建议 V1 就影响，否则广告只是静态页面。
  3. 是否要花真实店铺资金：建议要，否则没有经营取舍。
  4. 广告数据粒度：建议先按“游戏日”累计，不要按分钟级。
  5. 是否需要关键词：建议 V1 可以有关键词输入，但模拟逻辑先简化。
