# 时间变量说明

## 核心时间换算规则

| 规则 | 值 |
|------|-----|
| 1 周真实时间 = 1 年游戏时间 | 7天真实 = 365游戏日 |
| `REAL_SECONDS_PER_GAME_DAY` | `604800 / 365 ≈ 1656.16` 秒/游戏日 |
| `REAL_SECONDS_PER_GAME_HOUR` | `REAL_SECONDS_PER_GAME_DAY / 24 ≈ 69.0` 秒/游戏小时 |

---

## 变量说明表

| 变量名 | 类型 | 含义 | 时间轴 | 示例值 |
|--------|------|------|--------|--------|
| `run.created_at` | `datetime` | 对局创建时的真实时间，也是游戏时间的起点 | 真实时间 | `2026-04-10 07:59:42` |
| `run.manual_end_time` | `datetime` | 管理员手动设置的真实结束时间（已废弃作为游戏终点依据） | 真实时间 | `2026-05-08 07:59:42` |
| `run.total_game_days` | `int` | 对局总游戏天数，决定游戏时长 | 游戏日 | `1095`（对应真实约21天） |
| `run.duration_days` | `int` | 对局真实持续天数（冗余字段，以 `total_game_days` 为准） | 真实天 | `21` |
| `game_end_time` | `datetime` | 游戏终点时间 = `created_at + total_game_days × REAL_SECONDS_PER_GAME_DAY` | 真实时间 | `2026-04-10 + 21天 = 2026-05-01` |
| `current_game_tick` | `datetime` | 当前游戏进度对应的真实时间点 = `created_at + min(真实流逝秒数, max_real_seconds)` | 真实时间 | `2026-04-22 07:39:12` |
| `tick_time`（日志表） | `datetime` | 每次订单模拟对应的时间戳，以真实时间为单位递增 | 真实时间 | `2026-04-22 07:30:30` |
| `base_tick` | `datetime` | 本次模拟的起始时间 = 数据库中最新的 `tick_time`，或 `created_at` | 真实时间 | `2026-04-22 07:30:30` |
| `step_seconds` | `int` | 每次模拟推进的真实秒数 = `REAL_SECONDS_PER_GAME_HOUR × ORDER_SIM_TICK_GAME_HOURS` | 真实秒 | `552`（≈8游戏小时） |
| `missing_steps` | `int` | 待补跑的模拟步数 = `(current_game_tick - base_tick).total_seconds() // step_seconds` | — | `0` 表示已追上 |
| `elapsed_seconds` | `float` | 从 `created_at` 到现在的真实流逝秒数 | 真实秒 | `1,036,800`（12天） |
| `max_real_seconds` | `float` | 对局允许的最大真实秒数 = `total_game_days × REAL_SECONDS_PER_GAME_DAY` | 真实秒 | `1,813,495`（21天） |

---

## current_game_tick 计算逻辑

```
elapsed_seconds = now - run.created_at（真实秒数）
max_real_seconds = total_game_days × REAL_SECONDS_PER_GAME_DAY
clamped_seconds = min(elapsed_seconds, max_real_seconds)
current_game_tick = run.created_at + clamped_seconds
```

当真实时间超过游戏总时长时，`current_game_tick` 被 clamp 到 `game_end_time`，模拟停止。

---

## 常见误区

| 误区 | 正确理解 |
|------|----------|
| `tick_time` 是游戏内日历时间 | `tick_time` 是**真实时间戳**，每步推进约 69 秒 |
| `manual_end_time` 决定游戏终点 | 游戏终点由 `total_game_days` 决定，`manual_end_time` 仅供参考 |
| 延长对局只需改 `manual_end_time` | 必须同步修改 `total_game_days`，否则游戏终点不变 |
| `current_game_tick` 是游戏内年月日 | 是真实时间轴上的进度点，与 `tick_time` 同一时间轴 |
