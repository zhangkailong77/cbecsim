# CBEC SIM 新用户最小可用操作说明（MVP）

本文面向首次使用 CBEC SIM 的用户，帮助你以最短路径完成一局基础经营闭环，并了解关键注意事项。

## 1. 使用目标
- 完成从 `Step01 开局` 到 `Step05 店铺运营` 的最小流程。
- 能看懂资金变化：采购支出、物流/仓储支出、Shopee 回款、提现转入工作台。
- 遇到问题时可快速定位是“流程问题”还是“配置问题”。

## 2. 启动方式
- 一键启动前后端：`./start-dev.sh`
- 前端地址：`http://127.0.0.1:3001`
- 若登录时报 `ERR_CONNECTION_REFUSED :8000`，说明后端未启动。

## 3. 首次登录与开局
1. 进入登录页，使用账号密码登录。
2. 首次进入后在工作台点击“领取资金并开始”创建对局。
3. 开局后会看到：
- 对局状态（running）
- 目标市场（当前 MVP 主要为 MY）
- 当前游戏日（Day X）

## 4. 最小经营主流程（建议顺序）
1. `Step02 选品与采购`
- 从榜单选商品，下采购单。
- 采购会消耗工作台可用余额。

2. `Step03 国际物流与清关`
- 选择货代与清关方案并创建物流单。
- 物流费用会继续消耗可用余额。

3. `Step04 海外仓与入仓`
- 选择仓储策略并完成入仓。
- 仓储成本会计入支出。

4. `Step05 店铺运营（Shopee）`
- 下方为当前版本已实现的页面与能力（可直接操作）：

### 4.1 Shopee 页面入口与导航
- 从工作台点击 `进入运营（Step05）` 进入 Shopee。
- 左侧菜单可切换：
  - `我的产品`
  - `添加新商品`
  - `我的订单`
  - `我的收入`
  - `我的余额`
  - `银行账户`

### 4.2 添加新商品（已实现）
- 支持商品基础信息填写（标题、类目、描述等）。
- 支持规格字段与类目模板联动。
- 支持主图/副图/视频上传。
- 支持物流方式、价格、库存、限购等关键参数配置。
- 支持草稿保存、编辑后发布、上架/下架相关操作。

### 4.3 我的产品（已实现）
- 可查看已发布商品列表。
- 支持父商品与变体信息查看（含销量/库存）。
- 支持批量勾选与批量操作（按当前页面能力）。
- 支持进入商品详情并更新已上架商品信息。

### 4.4 我的订单（已实现）
- 支持按订单状态查看：待出货、运输中、已完成、取消等。
- 支持待出货“安排发货”和批量发货。
- 支持查看物流详情、手动推进物流节点（用于教学演示）。
- 支持查看订单结算信息。
- 系统已接入自动订单模拟与物流推进，订单会随游戏时间变化。

### 4.5 我的收入（已实现）
- 展示收入摘要（本周、本月、累计等）。
- 支持已入账与待入账查看。
- 支持按游戏时间口径进行筛选与统计。
- 订单完成后回款存在释放周期（当前已实现延迟释放规则）。

### 4.6 我的余额（已实现）
- 展示钱包余额与交易流水。
- 交易流水支持方向筛选（收入/支出）与订单号检索。
- 支持“提现”到工作台资金（需默认银行卡）。
- 提现成功后会在工作台资金侧体现为“提现转入”。

### 4.7 银行账户（已实现）
- 支持新增银行卡账户（姓名、银行、卡号）。
- 支持设置默认账户。
- `我的余额` 页面会读取默认银行卡信息。

### 4.8 Step05 建议操作顺序（新手）
1. 先到 `银行账户` 添加并设置默认卡（为后续提现做准备）。
2. 到 `添加新商品` 发布至少 1 个可售商品。
3. 到 `我的订单` 观察系统模拟订单并执行发货操作。
4. 到 `我的收入/我的余额` 查看回款与流水。
5. 在 `我的余额` 发起提现，再回工作台确认资金变化。

### 4.9 右侧提示/通知区（已实现）
- Shopee 页面右侧已接入提示/通知能力（通知抽屉）。
- 入口：右侧工具栏通知按钮。
- 当前用途：
  - 展示待处理发货订单提醒；
  - 红点数量与待处理订单数量联动（超过阈值按上限显示）；
  - 列表可看到订单号、买家、金额、倒计时、下单时间等关键信息。
- 使用建议：
  - 每次进入 Shopee 后优先看右侧提示区，先处理“待发货”再做其他运营操作；
  - 与“我的订单”页联动核对，避免遗漏超时风险订单。

## 5. 资金口径（重点）
### 5.1 工作台口径
- 仅统计“经营可用资金”：
  - 期初资金
  - 累计收入（仅提现转入）
  - 累计支出（采购/物流/仓储）
  - 当前可用余额

说明：Shopee 订单回款不直接计入工作台“收入展示”，需先在 Shopee 财务页提现后才转入工作台。

### 5.2 Shopee 财务口径
- Shopee 回款在 Shopee 财务页形成钱包余额（RM）。
- 可从“我的余额”发起提现，按固定汇率转入工作台（RMB）：
  - `1 RM = 1.74 RMB`

## 6. 提现操作说明（已实现）
1. 进入 `Shopee -> 我的余额`。
2. 确保已设置默认银行卡（无默认卡无法提现）。
3. 点击“提现”，输入金额（RM）。
4. 系统展示预计转入 RMB 金额，确认后提交。
5. 提现成功后：
- Shopee 钱包余额减少；
- 工作台可用余额增加；
- 工作台右侧可在“收入明细”看到“提现转入”记录。

## 7. 工作台资金明细（Tab）
右侧“阶段总览数据”支持 Tab：
- `收入明细`：仅显示提现转入
- `支出明细`：显示采购/物流/仓储逐笔记录

时间展示规则：
- 以游戏时间为主：`Day X HH:mm:ss`

## 8. 常见问题与排查
### 8.1 登录失败/服务不可用
- 现象：登录按钮报服务不可用、控制台报 `:8000/auth/login` 连接失败。
- 处理：确认后端已启动并监听 8000 端口。

### 8.2 为什么 Shopee 有回款，工作台收入没变？
- 原因：工作台收入只认“提现转入”，Shopee 回款需先提现。

### 8.3 提现按钮点不了
- 常见原因：
  - 未设置默认银行卡
  - 钱包余额为 0

### 8.4 提现金额提交失败
- 常见原因：
  - 提现金额 <= 0
  - 提现金额超过当前钱包可提现余额

## 9. 新用户建议操作路径（5分钟）
1. 开局创建对局。
2. Step02 下 1 笔采购单。
3. Step03 建 1 笔物流单。
4. Step04 建仓并入仓。
5. Step05 发布商品并查看订单/回款。
6. 在 Shopee 财务页提现一笔，回工作台查看收入与余额变化。

## 10. 当前版本边界（MVP）
- 以核心经营闭环与资金闭环为主，细节仍在迭代。
- 金融高级能力（审核、手续费、动态汇率、复杂风控）暂未覆盖。
- 若遇到显示口径疑问，以“右侧工作台资金 Tab + Shopee 财务页”双视角交叉核对。
